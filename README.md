# Install

For normal work need `node >= 6.0`

Install need modules^
```
npm install
```
Create temp database
```
cat ./sql/database.sql | sqlite3 ./cache/database.db
```


# Run Admin

From project root directory run:
```
./bin/admin
```

And open in your browser `localhost:9001`
